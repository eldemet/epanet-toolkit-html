[OPTIONS]

Purpose:
Defines various simulation options.

Formats:
UNITS
CFS/GPM/MGD/IMGD/AFD/
LPS/LPM/MLD/CMH/CMD
HEADLOSS
H-W/D-W/C-M
HYDRAULICS     
USE/SAVE filename
QUALITY
NONE/CHEMICAL/AGE/TRACE id
VISCOSITY
value
DIFFUSIVITY
value
SPECIFIC GRAVITY
value
TRIALS
value
ACCURACY
value
UNBALANCED
STOP/CONTINUE/CONTINUE n
PATTERN
id
DEMAND MULTIPLIER
value
EMITTER EXPONENT
value
TOLERANCE
value
MAP
filename
          

Remarks:
1.All options assume their default values if not explicitly specified in this section.  
2.Items offset by slashes (/) indicate allowable choices.  

Example:
[OPTIONS]
UNITS       CFS
HEADLOSS    D-W
QUALITY     TRACE  Tank23
UNBALANCED  CONTINUE  10
