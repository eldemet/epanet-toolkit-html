ENgetoption

Declaration:

int ENgetoption( int optioncode, float* value )
 
Description:

Retrieves the value of a particular analysis option.  

Arguments:

optioncode:
an option code (see below)
value:
an option value
 
Returns:

Returns an error code.  

Notes:

Option codes consist of the following constants:  
EN_TRIALS
0
EN_ACCURACY
1
EN_TOLERANCE
2
EN_EMITEXPON
3
EN_DEMANDMULT
4
