ENgetnodeid

Declaration:

int ENgetnodeid( int index, char* id )
     
Description:

Retrieves the ID label of a node with a specified index.  

Arguments:

index:
node index
id:
ID label of node
 
Returns:

Returns an error code.  
 
Notes:

The ID label string should be sized to hold at least 32 characters.  
 
Node indexes are consecutive integers starting from 1.
