ENgetlinkindex

Declaration:

int ENgetlinkindex( char* id, int* index )
     
Description:

Retrieves the index of a link with a specified ID.  

Arguments:

id:
link ID label
index:
link index
 
Returns:

Returns an error code.  
 
Notes:
 
Link indexes are consecutive integers starting from 1.
