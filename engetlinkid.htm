ENgetlinkid

Declaration:

int ENgetlinkid( int index, char* id )
     
Description:

Retrieves the ID label of a link with a specified index.  
 
Arguments:

index:
link index
id:
ID label of link
 
Returns:

Returns an error code.  
 
Notes:

The ID label string should be sized to hold at least 32 characters.  
 
Link indexes are consecutive integers starting from 1.
